
# Проект для собеседования QA-инженеров

## Задание
Проект построен на базе тренажера Калькулятор Факториала https://qainterview.pythonanywhere.com/.

Факториал - это произведение всех натуральных чисел от 1 до указанного числа включительно.
Математические правила
1. Факториал от 0 всегда 1
2. Факториал можно взять только от целого числа
3. Факториал можно взять только от положительного числа
4. Верхней границы по факториалу не существует

_Внимание! Тренажер содержит дефекты!_


## Структура проекта
Проект включает следующие фреймворки:
* Junit
* Selenide
* Selenium
* Cucumber
* Allure Report
---
* RestAssured

Задачи по Junit + Selenide|Selenium можно проверить в классе **AdvancedFactorialPageTest**.
**AdvancedFactorialPage** создан для описания страницы по паттерну Page Object.

Задачи по Cucumber ранжированы по уровню сложности:
01 - использует реализацию из **ExampleFactorialPageSteps**
03 - проверяет реализацию из **AdvancedFactorialPageSteps**


Файлы со словом _Example_ являются примером реализации тестовых заданий.


## Запуск тестов
Рекомендуется запуск из IDE!
Установите рекомендованные плагины по работе с Junit, Cucumber, Gherkin для упрощения.

Для запуска Cucumber тестов предварительно скомпилируйте проект через
`mvn clean validate compile`
И запустите функционал через IDE. 
Альтернативно вы можете использовать Cucumber CLI Runner для запуска.

Для запуска Junit тестов используйте функционал IDE или
`mvn clean validate compile test`

**Важно!** При старте проекта скачивается webdriver в папку **build**. По завершении, драйвер удаляется.

При падении теста в папке **reports** сохраняются скриншоты.

Опционально, отчеты доступны в виде отчетов Allure, сохранение происходит в **allure-reports**.
